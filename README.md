# Mini Gestionnaire de Réservations

Application de gestion de réservations de chambres d'hôtel développée avec Next.js et Tailwind CSS.

## Fonctionnalités

- Affichage de la liste des chambres (nom, prix/nuit, disponibilité)
- Réservation / Annulation d'une chambre
- Filtre pour afficher uniquement les chambres disponibles
- Compteur de chambres disponibles et réservées

## Technologies

- Next.js 16
- React (useState)
- Tailwind CSS

## Installation
```bash
git clone https://github.com/yanni-bit/reservation-app.git
cd reservation-app
npm install
```

## Lancement
```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.
